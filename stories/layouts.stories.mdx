# Layouts

Different layouts of the player

#### Default

<Canvas>
  <Story name="Default">
    <AudioPlayer src={SAMPLE_MP3_URL} />
  </Story>
</Canvas>

#### Width

Default width is `100%`, you can always overwrite it to any value that greater than `300px`.

<Canvas>
  <Story name="Width" height="700px">
    <div>
      <p>300px</p>
      <AudioPlayer style={{ width: '300px' }} src={SAMPLE_MP3_URL} />
      <p>400px</p>
      <AudioPlayer style={{ width: '400px' }} src={SAMPLE_MP3_URL} />
      <p>500px</p>
      <AudioPlayer style={{ width: '500px' }} src={SAMPLE_MP3_URL} />
      <p>600px</p>
      <AudioPlayer style={{ width: '600px' }} src={SAMPLE_MP3_URL} />
      <p>100%</p>
      <AudioPlayer src={SAMPLE_MP3_URL} />
    </div>
  </Story>
</Canvas>

  </Story>
</Canvas>

#### Custom icons

Pass custom icons as needed to the props `customIcons`. Below is an example to use custom play icon for the audio player.

```ts
CustomIcons {
  play?: ReactNode
  pause?: ReactNode
  rewind?: ReactNode
  forward?: ReactNode
  previous?: ReactNode
  next?: ReactNode
  loop?: ReactNode
  loopOff?: ReactNode
  volume?: ReactNode
  volumeMute?: ReactNode
}
```
